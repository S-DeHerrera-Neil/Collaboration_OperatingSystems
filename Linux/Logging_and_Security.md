# Understanding syslog, rsyslog, and syslog-ng Codes

| Code | Facility |
| ---- | -------- |
| 0 | kernel messages |
| 1 | user-level messages |
| 2 | mail system |
| 3 | system daemons |
| 4 | security/authorization messages |
| 5 | messages made by syslogd |
| 6 | line printer subsystem |
| 7 | network news subsystem |
